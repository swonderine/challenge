# Outlier Detection

A simply outlier detection in Python

## Getting Started

These instructions briefly tell you how to get this project running.

### Prerequisites

Project is created with Python 3.8.5.

#### Packages

* pandas: 1.2.3
* numpy: 1.20.1
* matplotlib: 3.3.4

Please make sure to have the above listed packages installed. Otherwise please install the packages according to your python3 environment, e.g:

```
$ pip3 install pandas numpy matplotlib
```

### Run the script 

Make sure to have git installed and copy the repository to your desired local folder

```
$ cd /path/to/yourlocalfolder
$ git clone https://github.com/swonderine/challenge.git

```

Before running the script make sure the file [train.gz](https://www.kaggle.com/c/avazu-ctr-prediction/data) is copied to the same folder where [challenge.py](https://github.com/swonderine/challenge/blob/main/challenge.py) sits. Run the script via `python3`

```
$ cd challenge
$ cp /path/to/train.gz .
$ python3 challenge.py
```

### Output

Two PNG images should be saved, showing the results of the two tasks.
