# Adverity Challenge

A simply outlier detection in Python

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Please make sure to have the used packages installed. Otherwise please install the packages depending on your python3 environment.

```
pip3 install pandas matplotlib numpy
```

### Run the script [challenge.py](https://gist.github.com/PurpleBooth/b24679402957c63ec426)

To run the script change to the directory where the script sits and run

```
python3 challenge.py
```
